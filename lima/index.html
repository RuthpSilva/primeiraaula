Bitt & Byte, Escola Técnica

O Curso Técnico de Desenvolvimento de Sistemas tem por objetivo habilitar profissionais para analisar requisitos funcionais e não-funcionais de produtos, desenvolver e testar sistemas de software, de acordo com as especificações do projeto, considerando as boas práticas do mercado de tecnologia da informação e as necessidades do usuário.

Daremos foco ao primeiro semeste, onde tivemos 4 matérias.

Hardware de Redes
  • Instalações de sistemas operacionais (como Windows e Linux);
  • Intalações de drivers e aplicativos dos periféricos;
  • Equipamentos passivos e ativo de rede;
  • Firewall.
Sistemas Operacionais
  • Arquiteturas dos sistemas operacionais;
  • Sistemas operacionais de código fechado e código aberto;
  • Editor de texto;
  • planilia eletronica.
Fundamentos de Programação Orientada a Objeto
  • Linguagem de programação;
  • Lógica de programação orientada a objeto;
  • Ambiente de desenvolvimento;
  • Controle de versões.
Linguagem de Marcação
  • Linguagem de programação;
  • Lógica de programação orientada a objeto;
  • Ambiente de desenvolvimento;
  • Controle de versões.
